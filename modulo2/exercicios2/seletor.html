Criando site com HTML

Aprendendo id

Usando id

Dentro da tag que você quer personalizar diferente, utiliza o id=" " e escolhe um nome pra essa tag.

Exemplo:< h3 id="principal" >. Na folha de estilo, você utiliza #principal ou h3#principal {} e coloca as configurações da tag.

Em HTML é id e em CSS é #.

Diferença id e class

Não se pode usar dois id com o mesmo nome, se quiser repetir a configuração de mais de uma tag, usar class. Escolher o nome da class e do id de acordo com a função, e nao pela forma, cor etc.

Class vem primeiro e id depois. O que sobrepõe é o id.

Aprendendo class

Usando class

Dentro das tags que você quer personalizar diferente, utiliza o class=" " e escolhe um nome pra essa tag.

Exemplo: < h2 class="subtitulo1" >. Na folha de estilo você utiliza .subtitulo1 {} e coloca as configurações da tag.

Em HTML é "class" e em CSS é ""."

Mais configurações do class

Da pra usar mais de uma classe dentro de um elemento. Exemplo: class = "class1 class2"

Também é possivel usar class e id no mesmo elemento. Exemplo: id="" class=""

Caso eu queira deixar um trecho do texto com as configurações de uma class, eu posso usar a tag span em volta da palavra e dentro dela colocar o nome da class
