CSS

CSS INLINE

Coloca style no body dentro da tag que quer estilizar, não é recomendado.

CSS INTERNO OU LOCAL

Coloca a tag style na head e abre um colchetes para a tag que quer estilizar.É aplicado para todas as tags com o mesmo nome na mesma página.

CSS EXTERNO

Coloca a tag link css no head e abre outro arquivo chamado style.css na mesma pasta
